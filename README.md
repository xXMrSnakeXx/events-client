# Event frontend

## You can run App using the command

`npm run install`

`npm run dev`

## Use of technology

1. React
2. formik
3. react-icons
4. react-intersection-observer
5. react-router-dom
6. yup
7. react-toastify
8. clsx
9. axios
10. react-loader-spinner

## In this app you can

1. Get all events
2. Add participants for event
3. Get all participants from event
